
Folder structure
-------------------

Concept Art		
	Containing concept graphical work on UI, individual objects, etc.

Game Docs		
	Containing descriptions, models and other files describing the scope, contents, requirements, ..., of the project.

MoonBase
	The technical project (eclipse) with PhoneGap.
	Because expertise on PhoneGap is low, the phonegap project is currently only aimed at windows8.
	An android version is developed in a separate repository: git@github.com:Josvanegmond/androidmoonbase.git
	The android version is native code.
